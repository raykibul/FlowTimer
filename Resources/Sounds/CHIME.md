# Completion Chime Sound

## Required File

### chime.mp3
- **Description**: A gentle, pleasant chime or bell sound to signal timer completion
- **Duration**: 2-5 seconds
- **Mood**: Gentle, non-jarring, satisfying

## Characteristics

The completion chime should be:
- **Gentle**: Not startling or harsh
- **Distinct**: Clearly different from ambient sounds
- **Pleasant**: A rewarding sound for completing a focus session
- **Brief**: 2-5 seconds maximum

## Suggested Sound Types

- Singing bowl / meditation bell
- Soft chime or wind chime
- Gentle gong
- Crystal bell
- Soft notification tone

## File Requirements

- **Format**: MP3 (128-192 kbps)
- **Duration**: 2-5 seconds
- **Sample Rate**: 44.1 kHz
- **Channels**: Stereo or Mono

## Usage in App

The chime plays once when:
- Timer reaches 0:00 (session completed)
- Focus mode is automatically disabled
- Completion message is displayed

The chime does NOT play when:
- User manually stops/cancels the timer
- App is quit during a session
