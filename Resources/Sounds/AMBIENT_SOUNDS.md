# Ambient Sound Files

## Required Files

Place the following MP3 files in this directory:

### 1. brook.mp3
- **Description**: Gentle brook or stream water sounds
- **Mood**: Peaceful, natural, calming
- **Suggested search terms**: "brook loop", "stream water ambient", "creek sounds"

### 2. ocean.mp3
- **Description**: Ocean waves crashing on shore
- **Mood**: Relaxing, rhythmic, meditative
- **Suggested search terms**: "ocean waves loop", "sea ambient", "beach waves"

### 3. white_noise.mp3
- **Description**: Pure white noise for focus
- **Mood**: Neutral, masking, concentration
- **Suggested search terms**: "white noise loop", "static noise ambient"

### 4. rain.mp3
- **Description**: Rain falling, gentle rainfall
- **Mood**: Cozy, calming, focused
- **Suggested search terms**: "rain loop", "rainfall ambient", "gentle rain"

### 5. forest.mp3
- **Description**: Forest ambience with birds and nature sounds
- **Mood**: Natural, peaceful, refreshing
- **Suggested search terms**: "forest ambient loop", "nature sounds", "birds forest"

### 6. coffee_shop.mp3
- **Description**: Coffee shop background ambience
- **Mood**: Social, productive, creative
- **Suggested search terms**: "coffee shop ambient", "cafe background", "restaurant chatter"

### 7. fireplace.mp3
- **Description**: Crackling fireplace sounds
- **Mood**: Warm, cozy, comfortable
- **Suggested search terms**: "fireplace loop", "fire crackling", "hearth sounds"

## File Requirements

- **Format**: MP3 (128-192 kbps)
- **Duration**: 2-5 minutes (will loop seamlessly)
- **Sample Rate**: 44.1 kHz
- **Channels**: Stereo preferred
